## 一页纸技能 (One-Page Skills)

本目录包含两个 Claude Code 一页纸汇报技能，适用于不同场景：

### 1. one-page (完整版)
- **名称**: ultimate-horizontal-timeline-architect
- **触发条件**: 当用户要求生成"向上汇报"、"项目一页纸"时
- **核心功能**:
  - 动态提炼模块主题
  - 同色系关键字高亮
  - **水平时间轴 (Horizontal Timeline) UI** - 当内容包含时间节点、里程碑或阶段性计划时，自动渲染为现代感十足的横向进度视觉
- **适用场景**: 项目汇报、进度追踪、里程碑展示

### 2. sample-one-page (简洁版)
- **名称**: elegant-one-pager-architect
- **触发条件**: 当用户要求整理沟通记录、会议纪要，或需要生成"一页纸汇报"、"高管总结"、"排版卡片"时
- **核心功能**:
  - 杂乱文本清洗为客观结论
  - 现代感、色彩丰富的 HTML 渲染
  - 使用渐变色区分板块
  - 视觉舒适的瀑布流布局
- **适用场景**: 会议纪要、沟通记录整理、高管汇报

---

**主要区别**:
- one-page: 侧重**时间轴可视化**，适合有明确里程碑的进度汇报
- sample-one-page: 侧重**内容提炼与快速呈现**，适合日常沟通记录整理
